# User Preferences

## 코딩 스타일
- 한국어 주석/커밋 메시지를 선호합니다.
- TDD 워크플로우를 준수합니다.
- 단순함을 우선하며 과도한 추상화를 지양합니다.

## UI/UX 선호
- 메인 컬러: DodgerBlue (#1E90FF) 또는 딥그린 (#2E8B57) 같이 차분하고 신뢰감 있는 색상을 사용합니다.
- Pretendard 폰트를 적용합니다.
- 채팅 가독성을 높이는 깔끔하고 미니멀한 레이아웃을 선호합니다.

## 기타
- 평균 응답 시간 100 ms 이하(동시 접속 1,000명 기준)를 목표로 합니다.
- 로컬 우선으로 개발하되, Docker/Kubernetes 배포 준비를 염두에 둡니다.
