# Project Objectives

## 노스스타 메트릭
- 실시간 채팅방에서 **1,000명 동시 접속** 지원 및 평균 메시지 지연 **100 ms 이하** 달성

## 입력 지표
- 주 3회 이상 부하 테스트 실행
- Kafka 토픽 메시지 처리량 모니터링(초당 메시지 수) 및 Redis 성능 지표 점검

## 마일스톤 목표

| 마일스톤 | 설명 | 목표 |
|----------|------|------|
| M0 | 프로젝트 셋업 | Nuxt/Spring/Kafka/Redis 개발 환경 구축 |
| M0.5 | 메시지 계약 정의 | 채팅 메시지 스키마 및 Kafka 토픽 설계 |
| M1 | 백엔드 구현 | Spring Boot로 Kafka 프로듀서/컨슈머 및 Redis 캐싱 구현 |
| M2 | 프론트엔드 구현 | Nuxt 3로 채팅 UI 및 socket.io 클라이언트 구현 |
| M3 | 통합 & 테스트 | end‑to‑end 통합 및 동시 접속 1,000명 부하 테스트 통과 |
| M4 | 배포 | Render 또는 Docker로 배포 및 모니터링 설정 |
