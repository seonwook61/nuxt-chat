# PRD: realtime-chat-1000

## 프로젝트 개요

**프로젝트명**: realtime-chat-1000
**목표**: 동시접속 1000명을 지원하는 실시간 채팅 시스템 구축
**기간**: 2026-01-18 ~ 2026-02-01 (2주)

## 핵심 요구사항

### 기능 요구사항

1. **실시간 채팅**
   - 채팅방 생성 및 입장
   - 실시간 메시지 송수신
   - 사용자 입장/퇴장 알림
   - 온라인 사용자 수 표시

2. **성능 요구사항**
   - 동시접속자: 1000명 이상
   - 메시지 지연: 100ms 이하 (p95)
   - 메시지 유실: 0%

3. **데이터 요구사항**
   - 최근 메시지 캐싱 (Redis, 10분 TTL)
   - 메시지 영구 저장 (선택사항, 향후 확장 가능)

## 제외 사항

- 사용자 인증/로그인 (선택사항, 간단한 닉네임만 사용)
- 파일 업로드
- DM (Direct Message)
- 메시지 검색
- 메시지 편집/삭제

## 성공 지표

1. 동시접속 1000명 테스트 통과
2. 메시지 지연 100ms 이하 (p95)
3. 무중단 운영 (24시간 stability test)
